# osumania_patterns_analyzer
A Python port for pattern analysis of [Interlude](https://github.com/YAVSRG/YAVSRG), adding preliminary LN pattern analysis on top of it.
[Interlude](https://github.com/YAVSRG/YAVSRG)键型分析算法的 Python 移植版本，在此基础上加入了LN键型的分析(初步)。

You can find the latest build in Actions.
你可以在Action里面找到最新构建。
